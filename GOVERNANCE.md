# Project Governance

NexCache follows a **Lead Developer Model**. 

## Decision Making
The project is founded and led by **Giuseppe Lobbene**. As the sole maintainer, Giuseppe is responsible for the technical direction, project oversight, and all final decisions regarding the NexCache architecture, code quality, and release cycles.

## Contributions
While the project is led by a single individual, community contributions are highly valued. Major architectural changes or new features should be discussed via GitHub Issues before implementation to ensure alignment with the project's vision.

## Evolution of Governance
As the project grows and attracts more contributors, this governance model may evolve into a more distributed committee-based system. Any changes to the governance will be documented in this file.
